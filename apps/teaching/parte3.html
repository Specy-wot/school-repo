JavaScript
Abbiamo visto come html e css riescono insieme a creare il corpo di un sito, ma la mente vera e propria risiede in javascript.
Nel 2020 ormai si può fare di tutto grazie a javascript, un singolo linguaggio ti prepara per creare siti, app, server, programmi etc Esso è uno degli aspetti più importanti dello sviluppo web dato che riesce a dare una funzionalità alla pagina oltre ad essere solo bella da vedere.

Con JavaScript si possono fare le seguenti cose:
  • Eseguire funzioni quando si preme un bottone
  • Modificare la pagina html in tutti i suoi aspetto
  • Creare, modificare, rimuovere elementi HTML
  • Prendere informazioni da server per poi mostrarli su html
  • Inviare informazioni al server o altro
  • Qualsiasi altra cosa un qualsiasi linguaggio può fare, di tutto

Eventi
Gli eventi sono ciò che invocano le funzioni javascript, ne esistono di moltissimi tipi ma quelli principali sono:

  • onClick (quando un elemento si preme)
  • onInput (quando si sta scrivendo in un elemento)
  • onChange (quando il conenuto di un elemento cambia)


Essi poi possono essere collegati alle funzioni javascript così che possano essere eseguite quando l'evento è invocato.

Funzioni e variabili
Funzioni e variabili sono le parti più fondamentali di qualsiasi linguaggio di programmazione, JavaScript ci aiuta moltissimo con le variabili, infatti non è come java dove ogni tipo di variabile va dichiarata in un tale modo e tale modo deve rimanere, su javascript qualsiasi variabile può avere qualsiasi tipo di valore, una stringa può diventare un array o un numero, un oggetto può diventare una stringa, di tutto.

Le funzioni invece esistono di due tipi:

  1. Funzioni normali
  2. Funzioni anonime


1) Le funzioni normali si scrivono con la struttura: "function nomeFunzione(parametri){ //codice }", esse possono essere chiamate da qualsiasi elemento che invoca un evento oppure all'interno del codice stesso. 2) Le funzioni anonime sono un concetto nuovo aggiunto da pochi anni e che probabilmente non sarà molto utile in questo momento ma è buono accennarle.
Esse sono funzioni, ma come dice il nome, sono anonime, esse non possono essere chiamate, invece sono usate a mo di variabile, oppure per accorciare una funzione, oppure principalmente da usare all'interno dei "callback" (cerca su google "javascript funzione callback"). Ricorda, questo non è fondamentale e verrà spiegato meglio in futuro.

Quando si esegue una funzione, si possono passare dei "parametri" che sono semplicemente dati che dai alla funzione così da poter essere usati.
Una funzione può come non, ritornare un valore, per ritornare un valore si usa la parola "return" e poi il dato che si vuole ritornare

Le variabili esistono di 2 tipi, "let" e "const", spesso vedrete una chiamata "var" ma è datata e NON VA usata.
let è il tipo di variabile normale, è utilizzata per inizializzare un nome e assegnargli un valore che poi può essere cambiato in futuro.
const è un tipo di variabile costante, cioè che una volta inizializzata, non può essere cambiato il proprio contenuto o tipo.

Una cosa molto importante da imparare dall'inizio è lo "scope" di una variabile, infatti una variabile può essere letta/scritta solo se è contenuta all'interno del proprio scope.
Vedetela come un albero, una foglia in basso non può accedere alle foglie in alto, ma le foglie in alto possono accedere alle foglie in basso.
Ogni "step" è fatto qualsiasi volta si utilizzano le { }, quindi per esempio all'interno di una funzione o di un if etc.

Assegnazione
Gli operatori di assegnazione rendono più corto l'assegnare una variabile e modificarne il contenuto.
Servono per non ripetere il nome della variabile due volte, rendendo il codice più facile da leggere.

  • = (assegnamento, NON CONFONDERE CON == o ===)
  • += (aggiungi e assegna)
  • -= (riduci e assegna)
  • *= (moltiplica e assegna)
  • /= (dividi e assegna)

Come detto prima, le variabili possono essere di tutti i tipi, ma questi tipi sono diversi l'uno dall'altro.
Di seguito ci sono tutti i tipi di tipi di variabile e il loro utilizzo:

  1. "ciao" (stringa)
  2. 1 (numero
  3. [] (array)
  4. {} (oggetto)
  5. true (booleano)


1) è una semplice stringa che contiene del testo al suo interno.
2) è un numero che contiene un valore numerale.
3) è una struttura dati per contenere vari valori al suo interno, viene poi iterato (si leggono gli elementi in ordine) o per qualsiasi altro scopo lo si vuole usare, esempio è : [1,2,3,"ciao",true] etc
4) è uno dei tipi più importanti, lo spiegherò in dettaglio in seguito, in pratica è una variabile che contiene altre variabili.
5) è un booleano true/false che può essere usato per le condizioni if/for/while etc


Oggetti
Gli oggetti sono un tipo di variabile molto speciale dato che possono contenere altre variabili o anche funzioni al porprio interno.
In javascript quasi tutto "sotto sotto" è un oggetto, infatti, per esempio, una stringa contiene varie funzioni integrate da javascript in se che possono essere usate su una variabile stringa direttamente, per esempio "ciao".toUpperCase() per renderlo tutto maiuscolo.

Un oggetto funziona a mo di chiave / valore, ogni chiave (chiamata key) in un oggetto, fa riferimento ad un valore.
Per poter accedere ad un valore, si deve fare: nomeVariabileOggetto.nomeChiave ed esso ritornerà il valore assegnato a quella chiave Esempio: 
        
    let oggetto = {
        chiave1: 1,
        chiave2: "ciao"
        chiave3: [1,2,3,"ciao",true]
        altroOggetto: {
           chiave1: 1
        }
    }

    console.log(oggetto.chiave1) //1
    console.log(oggetto.altroOggetto.chiave1) //ciao

Elemento per ID e modifica valore
Questo è l'ultimo argomento di oggi, la modifica del valore di elementi html tramite javascript.
Tramite javascript possiamo prendere tutti gli elementi presenti all'interno del codice html, lo possiamo fare in molti modi, ma se lo si deve fare per un singolo elemento specifico, la cosa migliore è usare un ID , ricorda l'ID è unico, un solo elemento DEVE averlo, in futuro vedremo anche come prendere tutti gli elementi tramite le proprie caratteristiche, come nome della classe, nome del tag etc.
Una volta preso un elemento tramite il proprio ID, possiamo metterlo all'interno di una variabile, questa variabile farà riferimento all'elemento html che andremo a leggere / modificare.
Ogni elemento ha molte funzioni e chiavi (oggetto) da poter leggere / scrivere per modificarne le caratteristiche, ma quelli che vedremo oggi sono solo per prendere un valore o metterlo.
Questi sono .innerHTML e .value, entrambi possono essere scritti e letti, se scritti, andranno a modificare il valore all'interno dell'elemento stesso.
